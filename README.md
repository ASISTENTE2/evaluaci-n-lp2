## Mi proyecto representa una fase crucial en la  implementación original en Java con Swing y persistencia JPA/Hibernate el enfoque actual de este proyecto es su modernización y migración hacia el ecosistema de Spring Boot, implementando una arquitectura web más robusta y escalable.


## Objetivos Clave de la Actualización

En esta etapa de desarrollo se busca transformar y optimizar la plataforma.

* **Migración a Spring Boot:** Transicionar la arquitectura base de la aplicación hacia el ecosistema Spring Boot para aprovechar sus ventajas en desarrollo rápido y configuración.

* **Adopción de Spring Data JPA:** Voy ah reemplazar la gestión de persistencia directa por el potente marco de Spring Data JPA, simplificando las operaciones de base de datos.

* **Interfaz de Usuario Web con Thymeleaf:** Implementaré una capa de presentación basada en web, utilizando Thymeleaf para ofrecer una experiencia de usuario moderna y accesible.

* **Gestión de Versiones Robusta con Git:** Aplicaré metodologías de control de versiones con Git y GitHub, incluyendo ramas de desarrollo, commits significativos y procesos de Pull Request para asegurar una colaboración eficiente y un historial claro del proyecto.

## Estructura del Proyecto

* `src/main/java`: Contiene la lógica de negocio y los componentes principales de la aplicación Spring Boot.
* `src/main/resources`: Alberga archivos de configuración, plantillas de interfaz de usuario (Thymeleaf) y scripts SQL de la base de datos.
* `pom.xml`: Define las dependencias del proyecto y su configuración con Maven.

## Base de Datos Reutilizada

La solución continuará utilizando la base de datos `bd2_yupanqui`.

El script completo para la creación de la base de datos y la inserción de datos de prueba está disponible en el archivo `bd_script.sql` ubicado en la raíz de este proyecto.


## Tecnologías Fundamentales

* Java 17+
* Spring Boot (Spring Web, Spring Data JPA, Spring DevTools)
* Thymeleaf
* Git / GitHub
* Base de Datos Relacional (especificaré el tipo de BD en los scripts)
